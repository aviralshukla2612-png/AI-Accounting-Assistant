# AI Accounting Assistant

## Features

- Upload CSV Files
- Upload Excel Files
- View Uploaded Data
- Search Customer Data
- Ask Business Questions

### Supported Questions

- Total Sales
- Top Customer
- Outstanding Invoices
- Monthly Trends

## Tech Stack

Frontend:
- React
- Vite

Backend:
- Node.js
- Express.js
- Multer
- XLSX

## Setup

### Backend

```bash
cd Backend
npm install
nodemon server.js
```

### Frontend

```bash
cd Frontend/frontend
npm install
npm run dev
```
